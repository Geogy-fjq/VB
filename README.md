# VB

## CashierSystem——基于VB.NET的超市收银系统  
### 实现的功能如下：  
（1）收银结算：每次在列表框中点击相应的商品时，会显示商品的名称、数量、单价折扣，并把用户所采购的所有商品在文本框内进行汇总，当输入实付款后单击“收款”按钮后计算应找的零钱。  
（2）信息管理：对于商品的种类、品牌商品、价格和折扣进行增删改查。  
（3）异常提示：当付款金额不合理（小于应付金额）时，给予提示；当对商品信息进行管理时，对非法操作进行提示。
